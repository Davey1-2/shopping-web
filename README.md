## Getting Started

First, ge tot backend and run the installation package:

```bash
cd src/backend
npm install
npm run dev
```

Then, run the development server:

```bash
cd ../ (only if u are still on backend directory from first step)
npm install
npm run dev
```

---

You should see your shopping list frontpage.
